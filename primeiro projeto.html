Boas-vindas ao meu site

"Neste exercício introdutório de HTML, você aprenderá a criar uma página simples utilizando os elementos essenciais da linguagem. Vamos explorar a estrutura básica de um documento HTML, incluindo as tags html, head, title e body, que formam a base de qualquer página web. Esse é o primeiro passo para começar a desenvolver sites e entender como os navegadores interpretam código para exibir informações aos usuários."

Esse exercício em HTML tem como objetivo principal ensinar os fundamentos da criação de páginas da web. Ele visa familiarizar o estudante com a estrutura básica de um documento HTML, introduzindo conceitos como tags, atributos e a hierarquia de elementos. A proposta é permitir que o aluno compreenda como o HTML funciona como uma linguagem de marcação, estruturando o conteúdo de forma lógica e acessível para os navegadores. Esses exercícios também ajudam a desenvolver habilidades práticas, como criar títulos, parágrafos, listas, links, imagens e tabelas, que são os blocos de construção de qualquer página web. Além disso, um exercício em HTML incentiva a criatividade e a resolução de problemas, oferecendo um espaço para experimentar diferentes formas de organizar e apresentar informações. Ao final, o objetivo é capacitar o estudante a construir páginas funcionais e visualmente organizadas, que servem como a base para projetos mais complexos no futuro, especialmente quando combinados com CSS e JavaScript.

  1. Definição- HTML (HyperText Markup Language) é a linguagem padrão para estruturar conteúdo em páginas da web.
  2. Estrutura Básica- Todo documento HTML começa com !DOCTYPE html e é estruturado com as tags principais; html, head, title, e body
  3. Principais Tags- h1 a h6: Títulos de diferentes níveis. p: Parágrafos. a: Links. img: Imagens. ul e ol: Listas (não ordenadas e ordenadas). table: Tabelas.li
  4. Atributos- Atributos fornecem informações adicionais às tags. Exemplo: a hrf="https://example.com>visite o site /a>
  5. Comentários- Ajudam a documentar o código e não são exibidos no navegador. !-- este é um comentario --
  6. Semântica no HTML- Tags semânticas, como header, footer, article e section, ajudam a organizar o conteúdo de forma mais significativa.
  7. Extensão- Arquivos HTML têm a extensão .html ou .htm.
  8. Compatibilidade- Funciona em qualquer navegador, sendo a base de toda página web.
  9. Evolução- O HTML5 é a versão mais recente, introduzindo novos elementos e melhor suporte para multimídia e interatividade.
  10. Complementos- HTML é frequentemente usado em conjunto com CSS (para estilização) e JavaScript (para interatividade).
